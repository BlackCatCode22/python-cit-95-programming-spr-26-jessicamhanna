# Builtin python functions Notes

## `print()`  -> displays output to the screen
## input()      -> gets user input (ALWAYS returns a string)

## int()        -> converts a value to an integer (whole number)
##                example: int("5") -> 5
##                example: int(3.9) -> 3  (truncates, does NOT round)

## float()      -> converts a value to a decimal number
##                example: float("3.5") -> 3.5
##                example: float(2) -> 2.0

## round()      -> rounds a number up or down
##                example: round(3.4) -> 3
##                example: round(3.6) -> 4
##                example: round(3.5) -> 4  (rounds to nearest even)

## type()       -> shows the data type of a value or variable
##                example: type(5) -> int
##                example: type(3.5) -> float
##                example: type("hi") -> str

## +            -> addition
## -            -> subtraction
## *            -> multiplication
## /            -> division (returns a float)
## //           -> integer (floor) division
## %            -> modulus (remainder)

## =            -> assignment operator (stores a value in a variable)
##                example: x = 5

## import math  -> gives access to math functions
## math.floor() -> rounds DOWN
## math.ceil()  -> rounds UP

# Reserved Words
## if       -> checks a condition and runs code if it is True
## else     -> runs code when the if condition is False
## elif     -> checks another condition if previous ones were False

## for      -> loops through a sequence (like a range of numbers)
## while    -> repeats code while a condition is True

## break    -> exits a loop immediately
## continue -> skips the rest of the loop and moves to the next iteration

## def      -> defines (creates) a function
## return   -> sends a value back from a function

## True     -> Boolean value meaning true
## False    -> Boolean value meaning false

## and      -> True if BOTH conditions are True
## or       -> True if at least ONE condition is True
## not      -> reverses a Boolean value

## in       -> checks if a value exists inside something
##            example: 3 in [1, 2, 3]

## None     -> represents no value / empty value

## import   -> brings in a module
##            example: import math

## pass     -> does nothing (placeholder for code)

## class    -> defines a class (object blueprint)
